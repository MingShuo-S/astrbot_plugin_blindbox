# CSV 导入功能使用指南

## 概述

BlindBox 插件支持通过 WebUI 批量导入任务数据，使用 CSV 文件格式。本指南将帮助您正确准备和导入 CSV 文件。

## 快速开始

### 1. 准备 CSV 文件

创建一个 CSV 文件，包含以下列（字段名可以是中文或英文）：

**必需字段：**
- `category` 或 `类别` - 任务分类
- `title` 或 `任务` - 任务名称

**可选字段：**
- `points` 或 `积分值` - 任务积分（默认为 0）
- `enabled` 或 `启用` - 是否启用（1/0 或 true/false，默认为 1）

### 2. CSV 文件示例

```csv
category,title,points,enabled
以智增慧,自习风险盲盒投资,10,1
以体强身,校园跑盲盒惊喜,10,true
以德润心,夸夸接龙暖心卡,10,1
以美立美,校园神奇动物通缉令,10,1
以劳励行,20元穷鬼美食探店盲盒,10,1
```

### 3. 编码要求

- **推荐编码**：UTF-8（可带 BOM）
- **支持编码**：UTF-8、GBK、GB2312、UTF-16 等
- 系统会自动检测文件编码并转换

### 4. 上传步骤

1. 打开 AstrBot WebUI
2. 导航到 "BlindBox CSV 导入" 页面
3. 点击 "选择 CSV 文件" 按钮
4. 选择您准备好的 CSV 文件
5. 点击 "开始导入" 按钮
6. 系统会显示预览，检查无误后点击 "确认导入"

## 常见问题

### Q: 上传失败，提示"没有找到上传的文件"

**A:** 请确保：
1. 选择了正确的 CSV 文件
2. 文件扩展名为 `.csv`
3. 文件大小不超过限制

### Q: 提示"无法解析文件编码"

**A:** 请尝试：
1. 用记事本打开 CSV 文件
2. 选择"另存为"
3. 编码选择 "UTF-8" 或 "UTF-8 with BOM"
4. 重新保存后再次上传

### Q: 部分任务未导入

**A:** 检查：
1. CSV 文件中是否有空行
2. 每行是否都有 category 和 title 字段
3. 查看上传后的错误提示信息

### Q: 如何导出当前任务列表？

**A:** 
1. 在 WebUI 中找到"任务管理"页面
2. 点击"导出 CSV"按钮
3. 下载的文件可以作为模板修改后重新导入

## 技术支持

如果遇到问题，请查看 AstrBot 日志获取详细错误信息：
- 日志位置：AstrBot 安装目录下的日志文件
- 关键日志：搜索 "CSV upload" 相关条目

## 附录：完整字段映射表

| 中文字段 | 英文字段 | 说明 | 必填 |
|---------|---------|------|------|
| 类别 / 种类 | category / tag / task_category | 任务分类 | 是 |
| 任务 | title / task_title | 任务名称 | 是 |
| 积分值 | points / task_points | 任务积分 | 否（默认0） |
| 启用 | enabled | 是否启用 | 否（默认1） |
